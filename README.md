# Orbit-OS
A Custom Firmware for the M17 handheld game console that pretty much is Android 2.3 on a Console. There is a App Store that works directly from the SD Card, it is called Play Store, even though back then it was called Android Market. This version of Android can be customized to have custom themes, (Installed From the Store Manually.) There is a Dark and a Light theme and by default when you setup the theme is white. There is a Setup process. you can set the date to show calender dates. change time for time, etc.

This is what apps there are.
Play Store - Installing apps, games, and custom themes, etc. has a bag with a colorful triangle icon
Settings - to show device info, change settings, etc.
Files app - shows everything on your SD Card, it can do simple functions like Copy, Paste, Cut, Move to, Copy to, etc.

Thats it. As you can see, its pretty BARE BONES! You can change this though and install things. I Know someone is about to ask me this: "Why are you using Android 2.3 and not a more recent version like Android 8 or 9?" Well heres your answer: The M17 console is simply to weak to run anything above 2.3. it was already a custom version and stripped. and Android 3 was for tablets only and required 512 MB RAM to run. Android 2.3 only needs 256 MB RAN to run smoothly and if there is no room for games, i will go down to Android 1.6, then Android 1.0, Enjoy!! Tell me experience in comments!
